# Frontend Application

This folder contains the frontend application used in the monitoring project.

The application is a simple web interface built using:

* HTML
* CSS
* JavaScript

It is used as a sample workload deployed in the Kubernetes cluster.

---

# Purpose

The application allows us to:

* Deploy a containerized application in Kubernetes
* Monitor application performance
* Generate metrics for Prometheus
* Visualize resource usage in Grafana

---

# Files

index.html – Main web page
style.css – UI styling
script.js – Simple frontend logic

---

# Deployment

The application is containerized using Docker and deployed to Kubernetes using the manifests in the `k8s` directory.
